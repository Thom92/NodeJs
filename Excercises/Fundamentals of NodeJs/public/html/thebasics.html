Basics of HTML, CSS and Javascript

HTML

HTML (Hyper text markup language) is the standard markup language for Web pages.
HTMl along with CSS and Javascript can make dynamic and immersive web pages.
Your code for the viewer to see has to been written the body tag of html.

CSS

CSS (Cascading style sheet) is one of the parts that HTML is missing, for without CSS, html is just plain boring text which no finese.
CSS is then used for styling websites through colors, moving text, making the more clear or blurrier.
The CSS files used for styling a web page can be used internal or external

Syntax of CSS

Selector − A selector is an HTML tag at which a style will be applied. This could be any tag.
Property − A property is a type of attribute of HTML tag. Put simply, all the HTML attributes are converted into CSS properties. They could be color, border etc.
Value − Values are assigned to properties. For example, color property can have value either red or #F1F1F1 etc.

Javascript

Javascript is the final part HTML needs to create a fun and interactive web page.
Javascript handles all variables, scripts, buttons, jsons objects, etc. which handles all the logic and interactive parts of a web page

Syntax of Javascript

var x, y, z; // How to declare variables
x = 5; y = 6; // How to assign values
z = x + y; // How to compute values
