Commands

There are several ways to interact with cmd and git which (if they are known) can make procceses faster and more efficient.
I´ve included some (in my opinion) noteworthy commands.

Commands for cmd

cd: is used for change a directory, making it easier to change workspace

mkdir dirname: is used to create a directory

Commands for git

git init - initiates git on the current directory

git add: can be used to add independent files, or everything with: git add .

git commit -m"message": is used to commit changes to the repository, along with a message, if you want to inform yourself (or others) of what you´ve pushed

git push: makes it possible to add your commit to github
