Jquery

What is jquery?

Jquery is a easy to use javascript library that simplifies:

  • HTML document traversing
  • Event handling
  • Animations
  • Ajax interactions

    • It has the motto of: Write less, do more.
    There are 2 ways that you can install jqery:
    The first is the Local installation where you download the library from: Jquery Download
    or you can include it in your html code with: Jquery CDN

    So why should we learn Jquery?

    Why you should learn jquery:
    1. It's lightweight
    2. It simplifies Javascript
    3. It also makes it easier to use Ajax calls and DOM (Document Object Modul) manipulation

    It also contains the following features:

    • CSS manipulation.
    • HTML event methods
    • Utilities

    Syntax

    The basic syntax of jquery is:
    $(selector).action()

    A $ sign to define/access jQuery
    A (selector) to "query (or find)" HTML elements
    A jQuery action() to be performed on the element(s)
