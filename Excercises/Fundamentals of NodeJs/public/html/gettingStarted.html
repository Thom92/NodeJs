Node.js Basics


What is NodeJs?


Nodejs is runtime environment for executing javascript.
It´s asynchrounous, which means it´s capable of handling a request, and execute another request, before the first request is done executing.
That makes it incredibly efficient if you are searching in a database. It is ideal for building applications which are:
  • Scaleable
  • Data-itensive
  • In protoyping

It will also be consitent with it´s code since it´s using javascript in both frontend and backend.

What can Node.js do?

Node.js can generate dynamic page content
Node.js can create, open, read, write, delete, and close files on the server
Node.js can collect form data
Node.js can add, delete, modify data in your database

What do you use Node for?

You can use for various things. Here are some examples:

  • Api
  • Webiste
  • Servers
  • Commandline programs
  • Scripts

It is recommended to not use Node for CPU intensive applications.
That could be scientific applicatiosn where you need to process a lot of data.

How do you use Node.js

You can use the Node Package Manager (npm) to manage install (and place) modules in a directory, so node can use them.
It also helps manage dependency conflicts.

Some of the more useful installations could be:

Nodemon to starting a server for testing
Express to use as a framework for web applications.
Budo to live reload your html pages.

How to use npm

You can use the npm directly in the commandline or through powershell.
The command: - npm install -g nodemon - will install nodemon globally, thanks to the "-g"
